# Backend Learning Journey 🚀

Welcome to my backend development learning repository. This project documents my progress as I build and explore modern backend technologies by creating a video-sharing platform inspired by popular streaming applications.

## 📚 About This Repository

This repository is dedicated to learning backend development from the ground up. It focuses on building scalable, secure, and production-ready APIs while following industry best practices.

The goal is to understand not just **how** to build backend applications, but also **why** certain architectural decisions are made.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/your-repo.git
```

### Navigate to the Project

```bash
cd your-repo
```

### Install Dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file in the root directory.

Example:

```env
PORT=8000
MONGODB_URI=your_mongodb_connection_string
ACCESS_TOKEN_SECRET=your_access_secret
REFRESH_TOKEN_SECRET=your_refresh_secret
ACCESS_TOKEN_EXPIRY=1d
REFRESH_TOKEN_EXPIRY=10d

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

### Run the Development Server

```bash
npm run dev
```
